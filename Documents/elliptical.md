$x$ = Point x on ellipse.
$y$ = Point y on ellipse.
$v$ = Height of ellipse.
$h$ = Width of ellipse.
$t$ = Angle of ellipse.
$m$ = Slope of line.
$a_1$ = x-intercept.
$b_1$ = y-intercept. 
$y = m * x + b_1$
$x / a_1 + y / b_1 = 1$
$y - y_1 = m * (x - x_1)$

$v^2 * x^2 * cos(t)^2 + 2 * v^2 * x * (m * x + b_1) * cos(t) * sin(t) + v^2 * (m * x + b_1)^2 * sin(t)^2 + h^2 * (m * x + b_1)^2 * cos(t)^2 - 2 * h^2 * x * (m * x + b_1) * cos(t) * sin(t) + h^2 * x^2 * sin(t)^2 - h^2 * v^2 = 0$
